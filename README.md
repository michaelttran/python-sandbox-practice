# python-sandbox

A personal sandbox for sharpening Python skills across three areas:

- **Data structures & algorithms** — LeetCode-style problems organized by pattern (hashmaps, linked lists, sliding window, stacks/queues, trees/graphs)
- **Low-level design** — OOP design pattern implementations (factory, etc.) with full requirements analysis
- **System design** — higher-level architecture practice

## Setup

```bash
source venv/bin/activate
pip install -r requirements.txt
```

Run any file directly:

```bash
python data-structures/hashmaps.py
python low-level-design/factory.py
```
